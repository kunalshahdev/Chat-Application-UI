# Chat Application UI

A chat application UI with a clean messaging interface. Features a modern design with message bubbles, contact list, and chat input.

## Features

- Realistic chat message bubbles (sent/received)
- Contact list sidebar
- Message input area
- Responsive layout
- Clean, modern design

## Running it

```bash
git clone https://github.com/kunalshahdev/Chat-Application-UI.git
```

Open `index.html` in any browser.

## Stack

HTML · CSS · Vanilla JavaScript
